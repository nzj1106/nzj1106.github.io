vue新建项目

Veröffentlicht am 2018-12-28 | Edited on 2019-11-21 | in Web技术 . 生活琐事

vue 项目的搭建 需要有node,npm等环境,接下来我会专门写一篇环境搭建的文章，在这里先忽略

首先，检查是否搭建好环境

WIN+R 输入cmd 打开命令窗口

$ node -v

$ npm -v

如果显示版本号说明安装成功

步入正题

项目初始化

1、全局安装vue-cli

$ npm install vue-cli -g

查看vue-cli是否安装成功

$ vue -V

如果显示版本号则说明可以新建项目啦

vue init webpack “项目名称”

执行完命令后会让你选择一些内容，如下图

$ npm install

$ cd 项目名称

$ npm run dev

显示 I Your application is running here: http://localhost:8080

浏览器中打开 http://localhost:8080 就可以看到你的项目啦
